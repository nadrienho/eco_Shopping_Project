EcoShop: Data-Driven Sustainable E-Commerce EcoShop is a full-stack e-commerce platform designed to combat "greenwashing" by providing users with transparent, verifiable environmental impact metrics. Unlike traditional retail sites, EcoShop quantifies the ecological cost of consumer goods using a proprietary "Eco-Score" algorithm, allowing users to align their spending with global climate targets.

The Mission Our project aims to bridge the "trust gap" in sustainable shopping. We replace vague marketing claims with empirical, data-driven insights into material composition and carbon intensity.

Key Features:

Eco-Score Algorithm: Real-time sustainability ratings based on material data.

Green Impact Dashboard: Personal analytics showing cumulative CO2 savings.

Verified Catalog: A product repository where "Must-Have" sustainability metrics are a prerequisite for listing.

Responsive UI: Optimized for a seamless shopping experience across all devices.

Technical Stack Layer Technology Frontend React (Powered by Vite) Backend Django Database Supabase (PostgreSQL) Deployment Vercel

Architecture & Methodology We followed the Agile Scrum methodology, developing the application in 1-week sprints over a 7-week timeline. This allowed for continuous integration between the Django API and the React frontend, ensuring high performance and data integrity.

Database: Relational schema managed via Supabase for real-time updates and secure authentication.

State Management: Optimized React hooks for managing shopping carts and environmental metrics.
